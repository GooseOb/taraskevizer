Usage: tarask [options] text
If text is not passed, interactive mode is enabled

EXAMPLES

Convert and latinize a word
	tarask --latin 'планета'
Prints "p{\fix l}a{\fix ne}ta"

Read from one file and write converted text to another
	tarask < ./cyr-text.txt > ./lat-text.txt

Enter interactive mode
	tarask
Prints "{\PREFIX} Enter the text:" and waits until you enter a new line

OPTIONS

General:
  -h --help
  -v --version

Alphabet:
  -l  --latin
  -lj --latin-ji
  -a  --arabic

When to replace і(i) by й(j) after vowels:
  -jr --jrandom
  -ja --jalways

Replace ґ(g) by г(h) in cyrillic alphabet:
  --h

Variations:
  -nv --no-variations
  -fv --first-variation

Mode (only one can be used):
  -html --html
  -abc  --alphabet-only

Other:
  -nec --not-escape-caps
  -nc  --no-color
  -st  --single-thread
