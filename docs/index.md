`taraskevizer` is a library and CLI that converts text written in the
academic orthography of the Belarusian language into the classical
orthography (Taraškievica).

- **Library** — import pipelines and configure conversion in JS/TS.
- **CLI** — convert text straight from the terminal (`tarask`).
- **Browser** — drop-in `<script>` bundle via jsDelivr.

Looking for JSON dictionaries? Check out the
[GitHub releases](https://github.com/GooseOb/taraskevizer/releases).
